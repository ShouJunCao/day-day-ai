📚 推荐学习资源

📄 必读论文

  • Attention Is All You Need
  • RAG (Lewis et al.)
  • ReAct (Yao et al.)
  • MemGPT (Packer et al.)
  • Memory in the Age of AI Agents

🛠️ 必学框架

  • LangChain / LangGraph
  • Mem0 / Letta / Zep
  • lm-evaluation-harness
  • vLLM
  • MCP (Model Context Protocol)

💻 必用工具

  • Cursor / Kiro (AI 编码)
  • Hermes Agent
  • Dify / Coze (低代码)
  • LangFuse (可观测性)

🔄 Java 经验转化

  • Spring Boot → 应用网关
  • MySQL/Redis → 向量DB/缓存
  • 并发编程 → Streaming
  • 设计模式 → Agent 架构
